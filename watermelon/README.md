# 西瓜分类 - BP神经网络项目

## 项目概述
本项目使用BP神经网络对西瓜数据集进行分类预测。通过构建多层感知器(MLP)模型，实现对西瓜品质的分类。

## 数据集
- 数据集来源：watermelon_3.csv
- 特征数量：8个
  - 密度
  - 含糖率
  - 纹理
  - 敲声
  - 色泽
- 类别：
  - 0：坏瓜
  - 1：好瓜
